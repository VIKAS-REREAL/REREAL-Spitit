## 5. FLOATING STATUS PILL (`src/ui/pill.py`)

### Specs
- Size: `120 × 34px` canvas, no title bar, no chrome
- Shape: fully rounded rectangle (pill shape), true transparent edges via `-transparentcolor` chroma key
- Draggable anywhere on screen, position saved to config
- Default position: bottom-right, `20px` from taskbar edge
- Always on top, `toolwindow=True`, `alpha=0.94`

### States & Animations

| State | Fill | Wave color | Animation |
|---|---|---|---|
| `hidden` | — | — | `withdraw()` |
| `recording` | `#FFD505` (yellow) | `#1a1a00` (dark) | Animated multi-sine waveform, 52ms tick |
| `transcribing` | `#111113` (dark) | `#4d9eff` (blue) | Spinning arc loader |
| `done` | `#111113` (dark) | `#3ddc84` (green) | Checkmark icon, hide after 900ms |
| `error` | `#111113` (dark) | `#ff5f5f` (red) | X icon, hide after 2800ms |

### Waveform
- 5 bars (not line) for the recording state: animated height bars, centered, rounded caps
- Each bar independently sine-animated at different frequencies (creates alive, organic feel)
- Bar heights: `[0.3, 0.6, 1.0, 0.6, 0.3]` base multipliers, modulated per tick
- Phase offset per bar: `[0, 0.4, 0.8, 1.2, 1.6]`
- Transcribing: single spinning arc, 2px stroke, `ACCENT_DIM` glow ring underneath

### Word count flash
After `done` state: briefly show word count `"12 words"` in `TEXT_SECONDARY` below the pill, fade out with the pill.

## 9. HOTKEY SYSTEM (`src/hotkey.py`)

### Architecture
- Use `keyboard` library for global hooks
- `HotkeyController` class with:
  - `hold_combo: str` — e.g. `"alt+left shift"`
  - `toggle_combo: str` — e.g. `"alt+left shift+space"`
  - `mode: Literal["hold", "toggle"]`
  - `on_start: Callable`, `on_stop: Callable`

### Recording Hotkey Combos (in UI)
When user clicks the hotkey button in settings:
1. Window captures ALL keyboard events temporarily
2. Tracks which keys are held — modifier keys noted separately
3. On any non-modifier keydown while modifier is held → record that as the combo
4. Display in real-time as user holds keys: `"Alt + LShift + ..."` 
5. On key release: finalize combo, show confirmation
6. Validate (see rules in section 6.3)
7. Escape cancels recording mode

## 11. PASTER (`src/paster.py`)

```python
def paste_text(text: str, cfg: dict) -> None:
    import pyperclip, keyboard, time
    from win10toast_click import ToastNotifier  # or plyer

    # 1. Always store for history
    # (handled in main.py before calling this)

    # 2. Clipboard (always copy, regardless of toggle — it's a fallback)
    pyperclip.copy(text)

    # 3. Auto-paste (if enabled)
    if cfg.get("output_paste", True):
        time.sleep(0.05)
        keyboard.press_and_release("ctrl+v")

    # 4. Notification (if enabled)
    if cfg.get("output_notification", True):
        _show_toast(text[:80] + ("…" if len(text) > 80 else ""))

    # 5. Sound (if enabled)
    if cfg.get("output_sound", True):
        _play_done_sound()

def _show_toast(message: str) -> None:
    # Use plyer for Windows toast notification
    # Non-blocking, fire-and-forget in thread
    import threading
    def _notify():
        from plyer import notification
        notification.notify(
            title="Spitit — Transcribed",
            message=message,
            app_name="REREAL - Spitit",
            timeout=3,
        )
    threading.Thread(target=_notify, daemon=True).start()

def _play_done_sound() -> None:
    import threading
    def _play():
        try:
            import sounddevice as sd
            import scipy.io.wavfile as wav
            from pathlib import Path
            sound_path = _get_sound_path()
            sr, data = wav.read(str(sound_path))
            sd.play(data, sr, blocking=False)
        except Exception:
            pass
    threading.Thread(target=_play, daemon=True).start()
```

## 15. DONE SOUND (`scripts/generate_sound.py`)

Generate a subtle 2-tone "done" chime:
```python
import numpy as np
from scipy.io.wavfile import write

sr = 44100
duration = 0.18  # seconds
t = np.linspace(0, duration, int(sr * duration))

# Two-tone chime: C5 then E5
freq1, freq2 = 523.25, 659.25
tone = (
    np.sin(2 * np.pi * freq1 * t) * np.exp(-t * 18) * 0.4 +
    np.sin(2 * np.pi * freq2 * t) * np.exp(-t * 22) * 0.3
)
tone = (tone * 32767).astype(np.int16)
write("assets/sound/done.wav", sr, tone)
```

## 23. BUILD SCRIPT (`build-release.ps1`)

```powershell
param([switch]$Installer)
$ErrorActionPreference = "Stop"
Set-Location $PSScriptRoot

Write-Host "Installing dependencies..."
python -m pip install --upgrade pip -q
python -m pip install -r requirements.txt -q
python -m pip install "pyinstaller>=6.0" -q

Write-Host "Generating icon..."
python scripts/build_icon.py

Write-Host "Generating done sound..."
python scripts/generate_sound.py

Write-Host "Running PyInstaller..."
python -m PyInstaller --noconfirm REREAL-Spitit.spec

$exe = "dist\REREAL-Spitit.exe"
if (-not (Test-Path $exe)) { throw "Portable build failed: missing $exe" }
Write-Host "Portable: $exe"

if ($Installer) {
    $iscc = $null
    foreach ($c in @(
        "${env:ProgramFiles(x86)}\Inno Setup 6\ISCC.exe",
        "${env:ProgramFiles}\Inno Setup 6\ISCC.exe",
        "C:\Program Files (x86)\Inno Setup 6\ISCC.exe"
    )) { if (Test-Path $c) { $iscc = $c; break } }

    if (-not $iscc) { throw "Inno Setup 6 not found. Install from jrsoftware.org" }
    & $iscc "installer\REREAL-Spitit.iss"
    $setup = Get-ChildItem "dist_installer" -Filter "REREAL-Spitit-Setup-*.exe" | Select-Object -First 1
    if ($setup) { Write-Host "Installer: $($setup.FullName)" }
}
```
