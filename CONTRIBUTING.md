# Contributing to REREAL - Spitit

Thank you for your interest in contributing to **REREAL - Spitit**! We welcome contributions that help improve the app and make it more efficient.

## How to Contribute

1.  **Fork and Clone**: Fork the repository to your own account and clone it locally.
2.  **Set up Dev Environment**:
    - Install Python 3.11+.
    - Install dependencies: `pip install -r requirements.txt`.
3.  **Make Changes**: Ensure your code follows the existing style. Keep files single-responsibility and avoid unnecessary comments.
4.  **Submit a PR**: Create a new branch for your feature or fix, and submit a Pull Request to the `main` branch.

## Code Style
- **Single Responsibility**: Each module/file should have one clear purpose.
- **Clean Code**: No unnecessary comments or dead code.
- **Review**: This project is under REREAL. Contributions are welcome but will be reviewed carefully to maintain quality and branding.
