# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-10

### Added
- Initial release of **REREAL - Spitit**.
- **Hold-to-Talk**: Hold `Alt + LShift` to record.
- **Toggle-to-Talk**: Press `Alt + LShift + Space` to start/stop recording.
- **Groq Whisper Integration**: High-speed, accurate speech-to-text.
- **Auto-Paste**: Automatically types transcribed text into the active window.
- **System Tray**: Runs in the background with a sleek tray icon.
- **Dark UI**: Premium dark-themed interface built with CustomTkinter.
- **Settings Window**: Manage API keys and configuration.
- **config.json**: Local storage for secure API key management.

### Notes
- Initial public release under the **REREAL** brand.
- Optimized for Windows desktop usage.
